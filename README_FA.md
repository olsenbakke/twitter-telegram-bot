# ربات تلگرام فالوور توییتر / X

بدون نیاز به API پولی توییتر. با RSS کار میکنه.
هر وقت اکانت هایی که اضافه کردی توییت جدید بزنن، مستقیم داخل تلگرام برات میفرسته. با متن + عکس.

---

### 1. ساخت ربات تلگرام
1. برو به @BotFather در تلگرام
2. بزن `/newbot` و اسم و یوزرنیم بده
3. توکنی که بهت میده رو کپی کن. یه چیزی مثل: `123456:ABC-DEF...`

### 2. نصب
```bash
pip install -r requirements.txt
```

یک فایل `.env` بساز و این داخلش بزار:
```
TELEGRAM_BOT_TOKEN=توکنی که از BotFather گرفتی
ADMIN_CHAT_ID=
```

### 3. اجرا
```bash
python twitter_telegram_bot.py
```

ربات روشن میشه.

### دستورات داخل تلگرام:
- `/start` - راهنما
- `/add vitalikbuterin` - اضافه کردن پروژه
- `/remove vitalikbuterin` - حذف کردن
- `/list` - لیست اکانت های دنبال شده

میتونی 50 تا اکانت همزمان اضافه کنی، مشکلی نداره.

### 4. هاست ۲۴ ساعته رایگان
برای اینکه همیشه روشن باشه، روی یکی از اینها رایگان دیپلوی کن:
- **Railway.app** - ساده ترین
- **Render.com**
- یا روی یک سرور VPS خودت

فقط کافیه همین 3 تا فایل رو آپلود کنی، و متغیر `TELEGRAM_BOT_TOKEN` رو ست کنی.

---
**نکات:**
- تاخیر چک کردن پیشفرض 90 ثانیه است. میتونی توی فایل `CHECK_INTERVAL` عوضش کنی.
- اکانت های خصوصی / Private کار نمیکنه.
- اگر یک اینستنس Nitter فیلتر بود، ربات خودش از بقیه امتحان میکنه. لیستش توی `NITTER_INSTANCES` هست.
- توییت ها با عکس هم ارسال میشن.
